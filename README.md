# POO_Artigo_10-08-2020
Realize a pesquisar de um artigo científico que tenha um dos temas abaixo:  Programação Orientada a Objeto ou Programação Estruturada ou Programação Funcional.  Faça a leitura do mesmo e responda aos três tópicos abaixo:  Qual objetivo do artigo?  Qual era a aplicação do tema pesquisado?  Qual foi a conclusão?  Ao criar o seu texto utilize o template disponível no conteúdo da aula do dia 10/08/20 (sbc_template.doc),  a pesquisa deve ser realizada pelo portal de periódico da CAPES, acesso também está no conteúdo da aula. Tarefa deve ser realizada de forma individual.
